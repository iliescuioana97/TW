Docs

Machines:

  • main page with all the available machines that you can access
  • you can always add a new one, by completing the Add Machine form

Node machine Home Page:

  • main page with statistics for the current machine

Node machine Users:

  • shows all users that have access on this particularly machine

Node machine File System:

  • shows the file system of the remote machine

Node machine Logs:

  • logs from the machine, showing the output from the executed commands

Node machine Console:

  • a shell prompt where you can execute commands and see an output message

Node machine Configs:

  • configure the services that you want to have on this machine

Node machine Processes:

  • shows the running processes on this machine, their PID, etc.
